## 3. SO SÁNH & TƯƠNG TÁC A-B

### 3.1. Bảng so sánh tổng quan

| Tiêu chí | Thái Cực A (Dev) | Thái Cực B (Đa Ngành) |
|----------|-----------------|----------------------|
| **Mục đích** | Đam mê, tài sản, brand | Cash flow, trang trải |
| **Audience** | Nhỏ, chuyên sâu, loyal | Rộng, phổ thông, volatile |
| **Chuyển đổi** | 1-3% (kén) | 5-10% (dễ) |
| **Thu nhập năm 1** | $0-500/tháng | $200-1000/tháng |
| **Thu nhập năm 3** | $2000-5000/tháng | $500-2000/tháng (bão hòa) |
| **Thời gian đến $** | 6-12 tháng | 1-3 tháng |
| **Bền vững** | Cao (moat: chuyên môn) | Thấp (dễ bị copy, trend) |
| **Rủi ro chính** | Chậm, có thể không bán được | Algorithm thay đổi, bão hòa |
| **Kiến thức cần** | Dev chuyên sâu | Content creation + marketing |
| **Thời gian/tuần** | 8-10 giờ | 10-15 giờ (volume) |
| **AI dependency** | Thấp (kinh nghiệm thật) | Cao (AI tạo hầu hết content) |

### 3.2. Cách A và B hỗ trợ nhau

```
A (Dev) ──────────────────► B (Đa Ngành)
│                              │
├──► Dev audience ────────────► Họ cũng mua sắm → Shopee affiliate
├──► Tech review skills ─────► Áp dụng review sản phẩm đa ngành
├──► Video editing skills ───► Dùng cho faceless content
├──► SEO knowledge ──────────► Áp dụng cho affiliate blog
└──► Email list dev ─────────► Segment: gửi tech deals, tool sales

B (Đa Ngành) ───────────────► A (Dev)
│                              │
├──► Cash flow ──────────────► Trang trải trong lúc A chưa có thu nhập
├──► Volume content skills ──► Áp dụng cho Shorts/Reels drive traffic
├──► Trend awareness ────────► Biết tool mới để review trên A
└──► AI tools mastery ───────► Dùng AI hỗ trợ script video dev
```

## 5. DECISION FRAMEWORK

### Khi nào focus A, khi nào focus B?

```
┌─────────────────────────────────────────────────────────────────┐
│  BẮT ĐẦU (Tháng 1-6)                                           │
│  ├─ A: 30% thời gian (foundation)                              │
│  └─ B: 70% thời gian (cash flow)                               │
│  Lý do: B nuôi A trong giai đoạn A chưa có thu nhập            │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  TĂNG TRƯỞNG (Tháng 7-12)                                      │
│  ├─ A: 50% thời gian (monetize + scale)                        │
│  └─ B: 50% thời gian (stabilize + automate)                    │
│  Lý do: A bắt đầu có thu nhập, cần tăng tốc                    │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  MATURE (Năm 2+)                                               │
│  ├─ A: 70% thời gian (compound, tài sản)                       │
│  └─ B: 30% thời gian (maintain, thuê VA)                       │
│  Lý do: A là tài sản dài hạn, B là side income                 │
└─────────────────────────────────────────────────────────────────┘
```
